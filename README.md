# Crowd-Funding-Dashboard
1.Project Overview
This is a Crowdfunding Dashboard created as a learning project using Tableau.
The purpose of this project is to understand how crowdfunding campaigns work and how data visualization can help in analyzing campaign performance.

2.What I Did in This Project
Cleaned and understood the dataset
Created KPIs to track campaign performance
Built interactive charts and dashboards in Tableau
Analyzed successful and failed crowdfunding campaigns

3.Dashboard Insights
Total number of campaigns
Successful, failed, and canceled projects
Comparison of goal amount vs pledged amount
Category-wise campaign performance
Country-wise campaign analysis
Backer count overview

4.Tools Used
Tableau
Excel / CSV

5.What I Learned
How to create dashboards in Tableau
How to analyze real-world data
How to present insights using charts and filters
Basics of data-driven decision making

6.Conclusion
This project helped me improve my data analysis and visualization skills.
It is part of my learning journey as a student / fresher and reflects my interest in becoming a data analyst.
